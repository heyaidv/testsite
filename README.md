# GREEN-API Тестирование

Веб-страница для тестирования методов GREEN-API WhatsApp.

## Описание

Этот проект представляет собой HTML-страницу с интерфейсом для тестирования основных методов GREEN-API:

- **getSettings** - получение настроек инстанса
- **getStateInstance** - проверка состояния инстанса
- **sendMessage** - отправка текстового сообщения
- **sendFileByUrl** - отправка файла по URL

## Функциональность

- Ввод параметров подключения (idInstance и ApiTokenInstance)
- Вызов методов API через удобный интерфейс
- Отображение ответов API в формате JSON
- Двухколоночный макет: поля ввода слева, ответы справа

## Как использовать

1. Откройте файл `index.html` в браузере
2. Введите `idInstance` и `ApiTokenInstance` из личного кабинета GREEN-API
3. Для методов `sendMessage` и `sendFileByUrl` заполните необходимые поля:
   - Номер телефона получателя (формат: 79991234567 или 79991234567@c.us)
   - Текст сообщения (для sendMessage)
   - URL файла (для sendFileByUrl)
4. Нажмите на соответствующую кнопку для вызова метода
5. Результат отобразится в правой колонке

## Требования

- Аккаунт в GREEN-API (бесплатный аккаунт разработчика)
- Созданный инстанс в личном кабинете
- Подключенный номер телефона через QR-код
